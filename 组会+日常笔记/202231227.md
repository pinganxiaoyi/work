可以调整 `TPaveText` 对象的位置和大小来缩小它在画布上的占用空间。`TPaveText` 的前四个参数定义了它在画布上的位置和大小，这些参数是相对坐标（范围从 0 到 1），分别对应左下角的 x 坐标、左下角的 y 坐标、右上角的 x 坐标和右上角的 y 坐标。

  例如，如果您希望 `TPaveText` 对象更小并且位于画布的其他位置，您可以这样调整参数：

  ```python
  pt = ROOT.TPaveText(0.6, 0.6, 0.8, 0.7, "blNDC")
  pt.AddText(f"Threshold: {threshold_value:.2f}")
  pt.SetFillColor(0)
  pt.Draw()
  ```

  在这个例子中，`TPaveText` 对象的位置被设置在画布的右上角，且相对较小。您可以根据需要调整这些参数以找到最适合您的展示需求的位置和大小。
